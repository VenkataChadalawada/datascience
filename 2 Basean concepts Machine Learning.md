# Bayes' Theorem

#### P(A|B) = {P(A) P(B|A)} /  P(B)

#### Eg: P(Spam/Free) = {P(Spam) P(Free|Spam)} / P(Free)

Assuming the presence of different entities are independent of each other - "Naive" Bayes Theorem
